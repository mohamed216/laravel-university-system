# نظام إدارة الجامعة المتكامل - Laravel University System

## 1. نظرة عامة على النظام

نظام إدارة جامعي متكامل Built with Laravel 12

### المستخدمين:
- 👨‍💼 المدير العام (Admin)
- 👨‍🏫 أعضاء هيئة التدريس (Professors)
- 👨‍🎓 الطلاب (Students)
- 👨‍💻 الموظفين (Staff)

---

## 2. الوحدات الرئيسية

### 2.1 إدارة الطلاب (Student Management)
- تسجيل الطلاب الجدد
- معلومات شخصية ومعلومات الاتصال
- الحالة الأكاديمية
- السجل الأكاديمي
- الشهادات والوثائق

### 2.2 إدارة أعضاء هيئة التدريس (Faculty Management)
- تسجيل الأساتذة
- التخصصات والمؤهلات
- جدول التدريس
- البحث والنشر

### 2.3 إدارة الكليات والأقسام (Faculties & Departments)
- الكليات (كلية الطب، كلية الهندسة، ...)
- الأقسام داخل كل كلية
- رؤساء الأقسام

### 2.4 إدارة المقررات (Course Management)
- المقررات الدراسية
- المتطلبات السابقة
- الساعات المعتمدة
- الكتب والمراجع

### 2.5 التسجيل والجداول (Enrollment & Schedules)
- تسجيل المقررات
- الجدول الأسبوعي
- القاعات والفصول
-冲突 الجداول

### 2.6 إدارة الدرجات (Grade Management)
- إدخال الدرجات
- حساب المعدل التراكمي
- التقديرات (A, B, C, ...)
- الشهادات

### 2.7 الحضور والانصراف (Attendance)
- حضور الطلاب
- حضور الأساتذة
- التقارير

### 2.8 المالية (Finance)
- الرسوم الدراسية
- الدفعات والفواتير
- الاستثناءات
- التقارير المالية

### 2.9 المكتبة (Library)
- الكتب والمصادر
- الإعارة والإرجاع
- الحجز

### 2.10 التقارير والإحصائيات (Reports & Statistics)
- تقارير أكاديمية
- إحصائيات الطلاب
- تقارير مالية
